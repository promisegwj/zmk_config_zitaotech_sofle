# zmk_config_zitaotech_sofle
zmk configuration for zitaotech sofle split keyboard

# 功能说明

## 1. 保留原Minilaair的习惯
把大拇指自然放的那个键和tab长按时设置为fn，fn加sedf是上下左右，fn加j是退格，fn加空格是回车。

## 2. 右手小红点三模式

### 2.1 默认层 - 鼠标指针模式
- 小红点控制鼠标指针移动
- **自动进入鼠标层**：移动小红点后自动切换到鼠标层
- **自动退出**：停止移动400ms后自动退出鼠标层

### 2.2 鼠标层 - J键滚动模式
- **进入方式**：移动小红点自动进入鼠标层，或手动切换到鼠标层
- **J键滚动**：按住J键，小红点变成滚动模式
  - 垂直滚动：上下推动滚动页面
  - 水平滚动：左右推动水平滚动
- **智能右键**：J键弹起时触发右键（仅在未滚动时）
  - 如果按住J期间移动过小红点（用于滚动），弹起时不触发右键
  - 如果按住J期间未移动小红点，弹起时触发右键
- **H键**：鼠标层H键为左键（`&mkp MB1`）

### 2.3 鼠标层按键
- **H键**：鼠标左键
- **J键**：滚动模式触发键（按住）/ 右键（弹起时，未滚动情况下）
- **M键**：cmd+r（刷新/运行）
y：回车
u：关闭
n：f13（用来语音输入）


## 3. 左手黑莓轨迹球方向键模式

### 3.1 默认功能
左手轨迹球作为方向键使用：上/下/左/右移动光标，方便选取文字。

- **节流控制**：
  - 首次触发：输出1格，立即响应
  - 冷却期：200ms内累计脉冲，不输出
  - 匀速输出：冷却期后以60ms固定间隔匀速输出
  - 重置：100ms无操作后重置状态


## 5. 层切换说明
- **层0（默认层）**：正常键盘输入，H键为普通H键
- **层1（功能层）**：F键、数字键盘、蓝牙控制
- **层2（鼠标层）**：鼠标控制、滚动、右键功能
